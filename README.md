# slidedeck

[whatis.enspiral.com](http://whatis.enspiral.com)

## how to

### install

```
git clone https://github.com/enspiral/slidedeck
cd slidedeck
npm install
```

### start

```
npm start
```

### develop

```
npm run develop
```

### test

```
npm test
```

### deploy

```
npm run deploy
```
